# Changelog

All notable public changes to LoCAL2 are documented in this file.

## Public Baseline — 2026-07-18

Baseline commit: `e8a5054`

This release marks the boundary between the previously developed public
LoCAL2 platform and subsequent private thesis-related research.

Included in the baseline:

- LLM-native tool selection
- ZeroMQ participant communication
- Episodic engram storage and score-weighted retrieval
- Critic scoring and feedback
- Cognitive persona support
- Evaluation and comparison harnesses
- Model, tool, memory and state-transition observability

Thesis-specific methods, datasets, ablation protocols, analyses and results
developed after this baseline are not included.

## 0.3.7 — 2026-07-06

- Published the `local2` Python package to PyPI
- Added packaged web interface
- Added command-line entry point
- Added local installation and setup workflow
